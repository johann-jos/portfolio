DisabledIn

DisabledIn is an innovative app designed specifically for people with disabilities, to find jobs in disabled friendly areas.

Key Features:

  • Customizable accessibility settings for various disabilities
  • Voice commands and screen reader optimization
  • Community features to connect with others
  • Resource directory for disability services
  • Simplified navigation with adaptive interfaces

Technologies Used:

React Native, TypeScript, Firebase, Accessibility APIs, Speech Recognition
